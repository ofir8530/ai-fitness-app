# AI Fitness & Nutrition Tracker 🥗💪

אפליקציית Web חכמה המיועדת לליווי ותמיכה בתהליך חיטוב וירידה במשקל, בעזרת בינה מלאכותית (AI).

## מה האפליקציה עושה?

- **ניתוח תזונה ב-AI:** צילום או תיאור הארוחה וקבלת ערכים תזונתיים אוטומטיים.
- **חישוב מטרות אישי:** הגדרת יעדי קלוריות וחלבון לפי נתוני המשתמש (גיל, משקל, רמת פעילות).
- **מעקב התקדמות:** מעקב אחרי משקל, היקפים ותמונות לאורך זמן.
- **צ'אט יועץ:** המלצות למתכונים ורעיונות לארוחות בהתאם לערכים שנותרו למשתמש באותו יום.

## טכנולוגיות (Tech Stack)

- **Frontend:** React / Next.js
- **Database:** Supabase
- **AI:** Google Gemini API
- **Styling:** Tailwind CSS
